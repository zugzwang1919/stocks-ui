Welcome

Welcome to my rather amateurish website. My goal here (and hopefully yours too) is to see whether or not I'm making meaningful stock selections or would just be better off buying a simple index ETF (like SPY) or Vanguard's S&P 500 Index Fund.

I have a ways to go here as working on this in the evenings and on weekends makes for slow progress. Ultimately, this site will be comprised of UI that is Single Page Application built around Angular. The backend will be a Spring Boot application. You can see what I've completed recently on the Recently Completed Page and what still needs to be finished on the To Do Page.

To play around on this site, you can either...

Create Your Own Account

...or...

Use a Shared Account

Ready to Give it a Try?

After logging in, you can create the securities that you plan to buy and sell using the "Stocks Being Tracked" pages.

Next, you should enter your BUYs and SELLs on the "Transactions" pages.

Finally, you will then be able to analyze the BUYs and SELLs vs benchmark transactions on the "Calculations -> Benchmark" page.

Caveats

  • This is most definitely a work in progress. Don't make any investment decisions without double-checking these calculations.
  • All of the returns are "simple" return calculations. On the wishlist... supporting Internal Rate of Return / Annualized Return calculations.
  • This website is running on an Amazon t3a.small EC2 instance. Please be tolerant of some poor performance. Maybe some day, I'll upgrade to a "real" server.
  • It's probably worth noting that I have primarily tested with Google's Chrome browser and a little with Mozilla's Firefox.

If you have any comments, feel free to contact me. And most importantly... Happy investing!
